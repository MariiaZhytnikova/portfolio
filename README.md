# Antihero Portfolio

## Overview
A modern, cinematic developer portfolio built with React, TypeScript, and styled-components.
Dark mode, design tokens, and clean architecture showcase skills and personal style.

## Tech Stack
- React 18 + Vite
- TypeScript
- styled-components (with design tokens)
- Framer Motion (animations)
- Tailwind optional for experimentation

## Folder Structure
src/
  app/
  theme/
  components/
  sections/

## Features
- Dark + accent color theme
- Reusable UI components (Button, Card, Container)
- Antihero naming for sections (Spotlight, Backstory, Arsenal, Signal)
- Responsive design & subtle motion
- Deployable to Vercel

## Roadmap
1. Foundation: Theme + GlobalStyle + Tokens
2. Layout: Navbar + Section wrapper
3. Sections: Spotlight, Backstory, Arsenal, Signal
4. Polish: Animations, responsiveness, accessibility
5. Deployment
6. Antihero flair: theme switching, gradient effects

## How to Run

``
npm install
npm run dev
``
